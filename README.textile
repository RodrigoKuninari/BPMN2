h1. *BPMN 2.0*

h2. _Descrição_

Mapeamento Relacional do BPMN 2.0
O trabalho consiste em estudar e entender a estrutura XML padrão adotada pela OMG para representar modelos de processos de negócio na notação BPMN 2.0.

h2. _Tecnologias/Linguagens Utilizadas_

* IDE: NetBeans  7.2.1
* Plataforma: Java Standard Edition 7
* Banco de Dados: PostgreSQL 9.1

h2. _Frameworks_

* Hibernate (JPA 1.0)
